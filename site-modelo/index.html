HTML

Nível de hierarquia das tags

<h1> </h1>
<h2> </h2>
<h3> </h3>
<h4> </h4>
<h5> </h5>
<h6> </h6>

Cada elemento h possui um nível de hierarquia informando aos mecaniscos de buscas a hierarquia de conteúdos da página, começando do h1 e indo até o h6.

Não se trata de tamanho de texto. No HTML o objetivo é dar valor semântico às tags. Uma vez que o tamanho é um ajuste na forma, essa alteração deve ser feita por meio do CSS.

Main

A tag main serve para os leitores de acessibilidade identificar o conteúdo principal da página.

Article

Envelopar conteúdo em um elemento article

article é outro elemento do HTML5 que adiciona significado semântico para a marcação. É um elemento de seção e é usado para envelopar conteúdos independentes. Funciona bem em blogs, fóruns ou artigos de notícias.

Considere o exemplo, se um livro é um article, então cada capítulo é uma section. Um article é utilizado para "standalone content" (conteúdo isolado ou um conteúdo como um todo), e uma section para agrupar conteúdos relacionados de acordo com a temática. Quando não houver relação entre os conteúdos, então deve-se utilizar uma div.

<div> - grupos de conteúdo
<section> - grupos de conteúdos relacionados
<article> - conteúdo independente

Header

Tornar a leitura da tela mais fácil

Utilizado para envelopar informação introdutória ou links de navegação para suas tags pais, e funciona bem em torno de conteúdos que se repetem no topo de multiplas páginas.

O header é para ser utilizado no body da página, que é diferente do head, que contém o título da página, meta informações, etc.

Nav

Tornar a navegação do leitor mais fácil com o elemento nav

Essa tag serve para envelopar os principais links de navegação na página (menu).

Se vários links forem utilizados no fim da página (rodapé) não é necessário marcá-los com a tag nav. Usar o footer neste caso será suficiente.
